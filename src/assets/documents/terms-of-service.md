# 1. การยอมรับเงื่อนไข
- คุณยอมรับและจะปฏิบัติตามข้อกำหนดและเงื่อนไขที่ระบุไว้ด้านล่างนี้ รวมถึงเงื่อนไขที่มีการเปลี่ยนแปลงหรือเพิ่มเติมใดๆ ซึ่ง Maseshi อาจเผยแพร่เป็นครั้งคราว (เรียกรวมกันว่า "เงื่อนไขการใช้งาน") **หากคุณไม่ยอมรับเงื่อนไขและข้อกำหนดทั้งหมดที่มีอยู่ในเงื่อนไขการใช้งานนี้ โปรดอย่าเข้าถึงหรือใช้งานเว็บไซต์และแอปพลิเคชันนี้**
- Maseshi อาจมีการเปลี่ยนแปลงข้อกำหนดการใช้งานนี้ในบางเวลา การที่คุณเข้าถึงหรือใช้เว็บไซต์และแอปพลิเคชันนี้ต่อไป ถือว่าคุณยอมรับการเปลี่ยนแปลงดังกล่าว การเข้าถึงและใช้เว็บไซต์และแอปพลิเคชันจะเป็นไปตามข้อกำหนดการใช้งาน กฎ และหลักเกณฑ์ฉบับปัจจุบันที่ประกาศทางเว็บไซต์และแอปพลิเคชัน ณ เวลาที่ใช้งานดังกล่าว โปรดตรวจสอบลิงก์ "ข้อกำหนดการใช้งาน" ที่หน้าแรกของ maseshi.web.app เพื่อดูข้อกำหนดปัจจุบัน ณ ขณะนั้น ถ้าคุณละเมิดข้อกำหนดการใช้งานใดๆ การอนุญาตให้เข้าถึงหรือใช้เว็บไซต์และแอปพลิเคชันนี้จะยุติลงโดยอัตโนมัติ

# 2. ทรัพย์สินทางปัญญา สิทธิ์ใช้งานแบบจำกัดแก่ผู้ใช้
- คุณสามารถเข้าถึงและใช้เว็บไซต์ ภายใต้เงื่อนไขที่คุณจะต้องปฏิบัติตามข้อกำหนดการใช้งาน ข้อตกลงด้านสิทธิ์ใช้งานกับ Maseshi และกฎหมาย Maseshi ยังคงเป็นเจ้าของแต่เพียงผู้เดียวในสิทธิ์ กรรมสิทธิ์ และผลประโยชน์ของไซต์ และสงวนสิทธิ์ทั้งหมดที่ไม่ได้ให้อย่างชัดแจ้งตามข้อกำหนดการใช้งานนี้ Maseshi สามารถแก้ไข เปลี่ยนทดแทน หรือยุติการดำเนินการไซต์หรือส่วนใดส่วนหนึ่งเมื่อใดก็ตาม ด้วยเหตุผลใดก็ตาม โดยแจ้งให้ทราบหรือไม่ก็ได้ โดยเป็นดุลยพินิจของ Maseshi แต่เพียงผู้เดียว Maseshi ให้บริการไซต์นี้แบบ "ตามสภาพ" และ "ตามที่พร้อมให้บริการ"

# 3. เครื่องหมายการค้าของ Maseshi
- ภายใต้วัตถุประสงค์ของเงื่อนไขการใช้บริการนี้ คำว่า "เครื่องหมายการค้า" จะหมายถึงกฎหมายจารีตประเพณีทั้งหมด หรือเครื่องหมายการค้าจดทะเบียน โลโก้ เครื่องหมายบริการ ชื่อทางการค้า ชื่ออินเทอร์เน็ตโดเมน หรือการบอกในลักษณะอื่นใดถึงแหล่งที่มาสินค้าในขณะนี้หรือในอนาคตที่ใช้โดย Maseshi
- ไม่มีส่วนใดที่ระบุไว้ในที่นี้จะมอบหรือจะถือว่ามอบสิทธิ์ให้คุณในการใช้เครื่องหมายการค้าของ Maseshi เว้นแต่จะกล่าวไว้โดยชัดแจ้งตามเงื่อนไขการใช้งานนี้
- คุณตกลงว่าจะไม่ใช้เครื่องหมายการค้าของ Maseshi ในลักษณะที่อาจสร้างความเสื่อมเสีย ดูหมิ่น หรือส่งผลในทางลบกับเครื่องหมายการค้าดังกล่าวหรือกับ Maseshi
- คุณตกลงว่าจะไม่โต้แย้งหรือคัดค้าน (เช่น ในการดำเนินการคดีทางกฎหมาย และอื่นๆ) หรือให้ความช่วยเหลือ หรือสนับสนุนบุคคลหรือนิติบุคคลอื่นใดก็ตามในการโต้แย้งหรือคัดค้านความถูกต้องสมบูรณ์ของเครื่องหมายการค้าของ Maseshi หรือสิทธิ์ในเครื่องหมายการค้าที่อ้างสิทธิ์โดย Maseshi

# 4. ข้อมูลที่คุณให้
- Maseshi (หรือบุคคลที่สามซึ่งดำเนินการแทน) อาจเก็บข้อมูลเกี่ยวกับการใช้ไซต์ของคุณ แพลตฟอร์มของบุคคลที่สามซึ่งคุณใช้เข้าถึงไซต์อาจเก็บข้อมูลเกี่ยวกับการใช้แพลตฟอร์มบุคคลที่สามนั้นๆ และทำให้ข้อมูลนั้นเข้าถึงได้โดย Maseshi เมื่อคุณยอมรับเงื่อนไขของแฟลตฟอร์มบุคคลที่สามนั้นๆ การเก็บและใช้ข้อมูลดังกล่าวทั้งหมดนั้นโดย Maseshi จะเป็นไปตามข้อกำหนดการใช้งานนี้ นโยบายความเป็นส่วนตัวของ Maseshi และกฎหมายที่มีผลบังคับ
- Maseshi จะใช้และปกป้องข้อมูลส่วนตัวของคุณ เช่น ชื่อและที่อยู่ ให้เป็นไปตามคำแถลงความเป็นส่วนตัวของ Maseshi ซึ่งประกอบด้วยเนื้อหาอ้างอิงตามเงื่อนไขการใช้งานเหล่านี้

# 5. เนื้อหาของคุณ
สำหรับภาพ วิดีโอ ข้อความ เสียง หรือเนื้อหาอื่นใดที่คุณอัปโหลดหรือโพสต์ไปยังไซต์ ("เนื้อหาของคุณ") คุณให้การรับรองและรับประกันว่า:
1. คุณมีสิทธิ์ที่จำเป็นทั้งหมดในการส่งเนื้อหาของคุณไปยังไซต์ และให้สิทธิ์ใช้งานตามที่ระบุไว้ในที่นี้
2. Maseshi ไม่จำเป็นต้องได้รับสิทธิ์ใช้งานจากบุคคลภายนอก หรือจ่ายค่าลิขสิทธิ์ให้แก่บุคคลภายนอกใด ๆ ซึ่งเกี่ยวข้องกับเนื้อหาของคุณ 
3. เนื้อหาของคุณไม่ได้ละเมิดสิทธิ์ใดๆ ของบุคคลภายนอก ซึ่งรวมถึงสิทธิ์ในทรัพย์สินทางปัญญาและสิทธิ์ความเป็นส่วนตัว
4. เนื้อหาของคุณสอดคล้องกับข้อกำหนดการใช้งานนี้และกฎหมายที่บังคับใช้ทุกประการร

# 6. การเชื่อมโยงเว็บไซต์ภายนอก
ในกรณีที่ไซต์สามารถใช้ได้ผ่านแพลตฟอร์มของบุคคลที่สาม หรือถ้า Maseshi จัดทำลิงก์จากไซต์ไปยังแพลตฟอร์มบุคคลที่สามใดๆ หรืออนุญาตให้บุคคลที่สามเชื่อมโยงจากแพลตฟอร์มของตนมายังไซต์ คุณรับทราบและยอมรับว่า Maseshi ไม่ให้การรับประกันใดๆ ทุกประเภท ทั้งโดยชัดแจ้งและโดยนัย และไม่มีส่วนรับผิดชอบต่อเนื้อหาหรือการปฏิบัติของบุคคลที่สามหรือแพลตฟอร์มดังกล่าว แพลตฟอร์มดังกล่าวไม่อยู่ในความควบคุมของ Maseshi และ Maseshi จัดให้มี และ/หรืออนุญาตให้มีลิงก์เหล่านี้เพื่อให้ความสะดวกแก่คุณเท่านั้น การมีลิงก์ไม่ถือเป็นการแสดงถึงความเกี่ยวข้อง การรับรอง หรือการยอมรับของ Maseshi

# 7. การจำกัดความรับผิดชอบ
Maseshi ไม่รับผิดชอบต่อความเสียหาย ความสูญเสีย หรือ ค่าใช้จ่ายที่เกิดขึ้นจากการใช้บริการของ Maseshi หรือ ระบบภายนอกที่เชื่อมโยงกับเรา โดยรวมถึงความไม่ถูกต้อง, ความล่าช้าของการรับส่งข้อมูล, ระยะเวลา หรือ เหตุขัดข้องของการเชื่อมต่อ ความเสถียรของระบบ จนถึง ไวรัส และ มัลแวร์คอมพิวเตอร์
